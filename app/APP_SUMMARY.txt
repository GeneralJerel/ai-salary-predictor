================================================================================
                            FEATURE ENGINEERING
================================================================================

The app includes intelligent feature engineering:

INPUT MAPPING:
- Experience Level: Entry → EN, Mid → MI, Senior → SE, Executive → EX
- Employment Type: Full-time → FT, Part-time → PT, Contract → CT, Freelance → FL
- Company Size: Small → S, Medium → M, Large → L
- Remote Ratio: 0-100 as slider value
- Job Title: Passed directly
- Location: Country code passed directly

FALLBACK SALARY CALCULATION:
1. Base salary = median salary for experience level + job title
2. Apply experience multiplier (EN: 0.7x, MI: 1.0x, SE: 1.15x, EX: 1.3x)
3. Apply remote bonus (5% per 100% remote)
4. Apply company size bonus (S: 0.9x, M: 1.0x, L: 1.1x)
5. Final = base × multiplier × remote_bonus × company_bonus
